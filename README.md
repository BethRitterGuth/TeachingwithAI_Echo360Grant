## Purpose of this Repository

This repository exists to:
- Provide teachers, instructional designers, and educators with high-quality, time-saving, creative activities they can immediately use or adapt.
- Demonstrate how AI can be responsibly used to enhance (not replace) human pedagogical expertise.
- Encourage open educational practices by sharing all activities under a permissive Creative Commons license (with non-commercial restriction to protect educators).

## Folder Structure

/subject-or-grade/           # e.g., /biology-grade-10, /ela-middle-school, /math-algebra-1
activity-title/
activity.md          # Main activity description (ready to copy-paste or print)
handouts/            Optional printable handouts, worksheets, cards, etc.
images/              Supporting images, diagrams, or AI-generated visuals
README.md            Quick overview & learning objectives
/contributors/               List of human curators and AI models used
/templates/                 Reusable prompts and templates for generating new activities


## How to Use These Activities

1. Browse or search for the topic/grade you need.
2. Open the `activity.md` file ‚Äî most activities are written in classroom-ready language.
3. Adapt as needed (and please share your improved versions back with the community!).
4. Always retain the original attribution and the CC BY-NC-SA 4.0 license notice when redistributing.
